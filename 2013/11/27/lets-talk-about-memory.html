Let’s talk about:memory

Recently on polish Facebook group JSNews, thanks to Zbigniew Braniecki, I found a very useful tool for monitoring memory in Firefox - about:memory.

You can open it by typing about:memory in your address bar. It has capabilities to measure memory usage at browser level, load and save dumps (as a gzipped JSON file), compare them and force garbage and cycle collection.

I strongly encourage you to dive into that tool, especially if you like me, thought that Firefox hasn’t got any tools for memory profiling. Well, not the first time I’m wrong .
